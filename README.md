# [Uro/python] Calculation for solving triangle

*Read this in other language: [**english**](https://github.com/Hanisekkus/Uro-Python_Reseni_Trojuhelniku), [**czech**](https://github.com/Hanisekkus/Uro-Python_Reseni_Trojuhelniku/blob/master/README.cz.md)*

 Task: Calculation for solving triangle from subject [URO] written in python with use of Tkinter.

## Instalation

### Python
 * Make sure you have installed [python](https://www.python.org/downloads/) on your device.
 
   > Tested on v.3.8.1

### Tkinter
 * Make sure you have installed Tkinter on your device.
 
   ```bash
   python -m tkinter
   ```

### Clone the project
 * In terminal:

    ```bash
    git clone https://github.com/Hanisekkus/Uro-Python_Reseni_Trojuhelniku.git
    ```

**_.. or_** 
 * In browser:

    [**Download**](https://github.com/Hanisekkus/Uro-Python_Reseni_Trojuhelniku/archive/master.zip)

## Usage

 * In terminal goes to the directory with project:

    ```bash
    py app.py
    ```
